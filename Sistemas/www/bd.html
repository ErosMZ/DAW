Creación de Base de Datos

A continuación, se muestra un ejemplo de cómo crear una base de datos y tablas en SQL, con explicaciones detalladas sobre cada paso.

Código SQL:

-- Crear una nueva base de datos
CREATE DATABASE MiBaseDeDatos;
USE MiBaseDeDatos;

-- Crear una tabla para usuarios
CREATE TABLE Usuarios (
    ID INT AUTO_INCREMENT PRIMARY KEY,
    Nombre VARCHAR(100) NOT NULL,
    Email VARCHAR(100) UNIQUE NOT NULL,
    FechaRegistro TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- Crear una tabla para pedidos
CREATE TABLE Pedidos (
    PedidoID INT AUTO_INCREMENT PRIMARY KEY,
    UsuarioID INT,
    FechaPedido TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    MontoTotal DECIMAL(10, 2) NOT NULL,
    FOREIGN KEY (UsuarioID) REFERENCES Usuarios(ID)
);

CREATE DATABASE se utiliza para crear una nueva base de datos llamada MiBaseDeDatos. Luego, USE selecciona esa base de datos para realizar las operaciones posteriores.

La tabla Usuarios almacena información sobre los usuarios. Cada usuario tiene un ID único, un Nombre, un Email único, y la fecha en que se registró (FechaRegistro).

La tabla Pedidos almacena información sobre los pedidos. Cada pedido tiene un PedidoID único, un UsuarioID que referencia al usuario que realizó el pedido, la FechaPedido, y el MontoTotal del pedido. El UsuarioID es una clave foránea que referencia el ID de la tabla Usuarios.

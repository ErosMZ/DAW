Recorrer un Array en JavaScript

A continuación, se muestra un ejemplo de cómo recorrer un array utilizando un bucle for en JavaScript. Este método es una forma sencilla y eficiente de iterar sobre los elementos de un array.

Código JavaScript:

// Definir un array de ejemplo
let miArray = [1, 2, 3, 4, 5];

// Usar un bucle for para recorrer el array
for (let i = 0; i < miArray.length; i++) {
    console.log(miArray[i]);
}

// Definir un array de ejemplo: Aquí se define un array llamado miArray con cinco elementos.

// Usar un bucle for para recorrer el array: Este bucle for inicia con i igual a 0 y se ejecuta mientras i sea menor que la longitud del array. En cada iteración, el elemento en la posición i del array se imprime en la consola.

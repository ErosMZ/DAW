Ejemplos de Bash y PowerShell

Bash - Pirámide de Asteriscos

#!/bin/bash

# Imprimir una pirámide de asteriscos
for ((i=1; i<=5; i++)); do
    for ((j=1; j<=i; j++)); do
        echo -n "* "
    done
    echo
done

Explicación de Bash

Bash es un intérprete de comandos de Unix y un lenguaje de scripting. Es ampliamente utilizado en sistemas Unix y Linux para ejecutar comandos del sistema y automatizar tareas.

Explicación de PowerShell

PowerShell es un shell de línea de comandos y un lenguaje de scripting desarrollado por Microsoft. Es utilizado principalmente en sistemas Windows para administrar y automatizar tareas del sistema.
